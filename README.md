# 🏋️ SkyFitnessPro

**SkyFitnessPro** — веб-приложение для онлайн-фитнеса, позволяющее пользователям выбирать курсы тренировок, отслеживать свой прогресс и заниматься в удобное время.

## ✨ Функциональность

### 🔐 Регистрация и авторизация
- Валидация email и пароля
- JWT токены для аутентификации
- Защищенные маршруты

### 📚 Каталог курсов
- Просмотр всех доступных программ тренировок
- Детальная информация о каждом курсе
- Поиск курсов по названию

### 📦 Управление курсами
- Добавление курсов в личную коллекцию
- Удаление курсов из коллекции
- Просмотр прогресса по каждому курсу

### 🎓 Тренировки
- Просмотр списка тренировок курса
- YouTube видео для каждой тренировки
- Список упражнений с целевыми показателями

### 📈 Отслеживание прогресса
- Возможность отмечать выполненные упражнения
- Визуализация прогресса (проценты выполнения)
- Сброс прогресса при необходимости

### 🎨 Особенности интерфейса
- Адаптивный дизайн для мобильных устройств
- Модальные окна для авторизации и уведомлений
- Toast-уведомления для обратной связи с пользователем
- Система подтверждения действий (AlertModal)

## 🚀 Установка и запуск

**Клонируйте репозиторий**
```bash
git clone https://github.com/yanakrtvn/SkyFitnessPro
cd SkyFitnessPro
Установите зависимости

bash
npm install
Настройка API

По умолчанию API endpoint настроен на https://wedev-api.sky.pro/api/fitness.
Если нужно изменить, отредактируйте API_BASE_URL в src/api/core/FitApi.js.

Запуск в режиме разработки

bash
npm run dev
Приложение будет доступно по адресу http://localhost:5173

🎯 Особенности реализации

Централизованная обработка ошибок API
Понятные сообщения для пользователя на русском языке
Разграничение ошибок валидации и серверных ошибок
Мгновенная обратная связь через Toast-уведомления
Подтверждение важных действий (добавление/удаление курсов, сброс прогресса)
Адаптивный дизайн для всех устройств
Прогресс-бары для визуализации достижений
